連上 GitHub 的步驟
目前系統找不到 git 指令，請先完成下面步驟：
1. 安裝 Git（若尚未安裝）
到 https://git-scm.com/download/win 下載並安裝 Git for Windows
安裝時可勾選「Add Git to PATH」
安裝完成後重新開啟終端機（或 Cursor）
2. 在本機初始化並推送到 GitHub
在終端機中執行（請把 您的使用者名稱 和 您的倉庫名稱 換成實際名稱）：
cd C:\Users\user\Desktop\resume# 初始化 Gitgit init# 加入 README.mdgit add README.md# 第一次提交git commit -m "Add README with name and skills"# 連到 GitHub（請先在 GitHub 網站上建立一個新的空倉庫）git remote add origin https://github.com/您的使用者名稱/您的倉庫名稱.git# 推送到 GitHub（主分支名稱可能是 main 或 master）git branch -M maingit push -u origin main
3. 在 GitHub 建立倉庫
登入 https://github.com
點「New repository」
倉庫名稱可設為 resume（或您喜歡的名稱）
不要勾選「Initialize with README」（因為您本地已有 README.md）
建立後，把上面 git remote add origin 的網址改成這個倉庫的 HTTPS 網址
4. 若使用 SSH 或需要登入
使用 HTTPS 時，第一次 git push 會要求 GitHub 帳號與密碼（或 Personal Access Token）
若想用 SSH，需先在 GitHub 設定 SSH key，並把 origin 改成 git@github.com:您的使用者名稱/您的倉庫名稱.git


cd C:\Users\user\Desktop\resume

# 初始化 Git
git init

# 加入 README.md
git add README.md

# 第一次提交
git commit -m "Add README with name and skills"

# 連到 GitHub（請先在 GitHub 網站上建立一個新的空倉庫）
git remote add origin https://github.com/您的使用者名稱/您的倉庫名稱.git

# 推送到 GitHub（主分支名稱可能是 main 或 master）
git branch -M main
git push -u origin main
