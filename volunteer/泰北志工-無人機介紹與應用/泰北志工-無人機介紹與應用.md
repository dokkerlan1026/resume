# 泰北志工服務 - 無人機介紹與應用

**活動類型**：海外資訊志工服務  
**服務對象**：泰北地區學生  
**擔任角色**：講師 / 課程設計  

## 📋 活動概述
在泰北資訊志工服務計畫中，我規劃並執行了「無人機介紹與應用」課程。透過無人機的實機展示與基礎觀念講解，拓展當地學生的科技視野，並探討無人機在現代社會的多元應用情境，例如農業、空拍與無人機物流等。

## 🛠 課程涵蓋技術與工具
| 分類 | 內容 |
| --- | --- |
| 核心主題 | 無人機飛行原理、安全規範、產業應用案例 |
| 教學工具 | 小型展示無人機、教學簡報、影片輔助素材 |
| 相關技能 | 科技科普推廣、動態設備展示、跨文化溝通 |

## ✨ 主要活動內容

### 基礎知識介紹
- 無人機基本構造與運作原理
- 操作安全規範與注意事項

### 產業應用分享
- 介紹無人機在生活中的應用（物流、空拍、勘災等）
- 播放相關案例影片，引發學生學習興趣

### 實機展示與體驗
- 安全範圍內的實機飛行展示
- 基礎的操作示範與學生初步體驗引導
- 問答環節與技術交流

## 🎯 核心成就與影響
- [x] 成功將新興科技（無人機）以淺顯易懂的方式介紹給當地學生
- [x] 透過實機展示大幅提升學生的參與度與專注力
- [x] 啟發學生思考科技如何改善生活與在地農業

## 💡 關鍵學習點
- **安全控管**：在教學環境中進行飛行設備展示的風險管理與安全宣導
- **科普表達**：將技術術語轉化為日常語言的表達技巧
- **臨場反應**：設備展示過程中的突發狀況排除
