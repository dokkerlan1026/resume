# 泰北志工服務 - 網頁設計教學

**活動類型**：海外資訊志工服務  
**服務對象**：泰北地區學生  
**擔任角色**：講師 / 課程設計  

## 📋 活動概述
在此次泰北資訊志工服務中，我負責教導當地學生基礎的網頁設計概念。透過實際操作演練，讓學生從零開始體驗網頁開發的過程，並培養他們對於資訊科技的興趣與實作能力。

## 🛠 課程涵蓋技術與工具
| 分類 | 內容 |
| --- | --- |
| 核心主題 | HTML 結構、CSS 樣式整合 |
| 實作工具 | 網頁編輯器、瀏覽器開發者工具 |
| 相關技能 | 課程規劃、實機操作指導、跨文化溝通 |

## ✨ 主要活動內容

### 網頁基礎概念
- 認識網頁的組成與運作原理
- HTML 基礎標籤介紹（標題、段落、圖片、連結）

### 實作演練
- 建立人生第一個 HTML 檔案
- 使用 CSS 進行基礎版面美化（顏色、字體、排版）
- 實機操作指導，協助學生解決程式碼錯誤

### 成果展示
- 學生完成個人化網頁設計
- 互相觀摩與分享設計理念

## 🎯 核心成就與影響
- [x] 讓學生建立撰寫程式的成就感
- [x] 成功帶領全班完成個人靜態網頁作品
- [x] 激發學生對資訊科技領域的興趣

## 💡 關鍵學習點
- **教學引導**：初學者面對程式碼的引導技巧
- **耐心與溝通**：如何在多語言環境下進行有效的雙向溝通
- **資源應用**：運用離線資源進行網頁教學的策略
