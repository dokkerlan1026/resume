# 地下道淹水辨識

**開發類型**：專案開發  
**技術特色**：影像辨識、LINE Notify 即時通報  

## 📋 專案概述
本專案旨在透過影像辨識技術，即時監控地下道水位狀況，以防範積水或淹水帶來的交通與生命財產風險。透過系統的自動偵測，一旦水位超越安全標準，能夠即時發送警報並通知相關單位與用路人，保障交通安全並提升通行效率。

## 🛠 技術棧
| 分類 | 技術 |
| --- | --- |
| 影像處理 | 影像辨識、顏色/高度判斷機制 |
| 通報系統 | LINE Notify API |
| 標示設計 | 虛擬標線尺與三色水位高度定義 |

## ✨ 主要功能

### 自定義預警水位辨識
系統定義三階段水位高度警戒與對應顏色，以利視覺化判斷與系統識別：
- **綠色高度**：40cm 以下（安全範圍）
- **黃色高度**：41cm - 60cm（警戒範圍）
- **橘色高度**：61cm 以上（危險範圍，需即時通報）

### 自動通報機制
- 結合 **LINE Notify** 進行成果展示與系統實作。
- 即時偵測水位變化，若達到淹水標準，即主動向指定的群組或使用者發送通報。

## 🎯 預期效益
- [x] **減少財產損失**：降低車輛與相關設施受損的風險。
- [x] **提升交通安全**：減少因淹水引起的意外事故，保障行人及車輛通行安全。
- [x] **促進通行效率**：第一時間掌握淹水資訊，避免淹水造成交通中斷或擁堵。
- [x] **資料蒐集與分析**：系統運作所記錄的水位數據，可作為未來城市防災規劃的參考依據。

## 💡 未來展望與計畫
- **3D 數位轉換與精準化**：將標線尺上的數字轉換為 3D 模型，利用數位化技術提高數據可視化與準確性。透過更清晰易讀的水位高度數值做為判斷依據，以便後續分析應用。
- **法規與因地制宜**：依照各地相關法規、條例及標準，對預警標準進行必要的調整，滿足當地防災、水資源管理等不同領域的需求。
- **系統整合**：未來可結合地理資訊系統（GIS）或大數據處理工具，進一步提升系統適用性與效能。
- **環境適應力優化**：針對不同區域的特殊環境條件（如降雨量變化趨勢、地形特徵、人為干預因素等）進行模型訓練與優化，提供更精確有效率的水位預警與災防管理方案。
