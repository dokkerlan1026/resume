# 憨吉 1.0：AI 學伴系統初探

**專題性質**：AI 教育科技專題（原型驗證）  
**團隊成員**：吳逸宣、曾俊豪、廖健安  
**指導老師**：簡子超  

## 📌 專案簡介
憨吉 1.0 是以「學習同伴」為核心概念的 AI 學伴系統初版，透過大型語言模型（LLM）與檢索增強生成（RAG）技術，讓學生可將個人筆記轉為可檢索知識庫，並在對話中進行教學互動與知識整理。

## 🎯 背景與動機
- **學習同伴**可促進知識分享、提升動機與深化理解。
- **教中學**能幫助學生在講解過程中找出盲點、強化理解。
- 將 AI 設計為可被教導的夥伴，降低互動門檻並提升學習主動性。

## 🧠 核心技術
- **大型語言模型（LLM）**：提供自然語言理解與生成能力。
- **檢索增強生成（RAG）**：先檢索再生成，降低模型幻覺。
- **向量資料庫**：儲存與比對學生筆記向量，支援語意檢索。

## 🛠 實作技術（簡報列示）
- Python
- FastAPI
- PostgreSQL（向量資料庫）
- Ollama
- Groq Cloud
- RAG 流程整合

## 🧩 系統流程摘要
- 學生輸入筆記與知識，建立個人知識庫。
- 系統將內容向量化並存入資料庫。
- 對話時依問題檢索相關知識，再由 LLM 生成回應。
- 讓學生在「教導 AI」與「與 AI 對話」中反覆內化知識。

## 🧪 入班測試與結果
- **導入課程**：慈濟大學資管系大一「資訊科技與管理概論」。
- **參與人數**：19 位學生，每位學生配備獨立憨吉 AI 學伴。
- **問卷結果**（22 題，滿分 5 分）：
  - 整體平均：**4.23**
  - 學習模式滿意度（9 題）：**4.2**
  - 科技接受度（13 題）：**4.3**
- 學生普遍回饋系統有助於複習、抓重點、整理筆記與加深印象。

## ✅ 結論
本研究完成一套可教導、可知識統整、具個人化互動能力的生成式對話學習系統。透過 LLM + RAG + 向量資料庫的整合，憨吉 AI 學伴可支援學生主動建構知識，並在互動中提升理解深度與學習成效。

## 🚀 未來展望

- **使用者體驗優化**：提升對話流暢度與介面友善性。
