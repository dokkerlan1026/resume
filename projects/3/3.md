# 專案標題

**開發類型**：個人自主專案  
**技術特色**：PHP + LAMP 環境  
**評價**：獲老師高度評價  

## 📋 專案概述
此為自學開發的線上訂餐平台，採 PHP 實現，提供完整的訂餐流程。系統涵蓋會員管理、菜單展示、購物車、訂單處理與付款流程，部署於 LAMP 環境運行穩定。

## 🛠 技術棧
| 分類 | 技術 |
| --- | --- |
| 後端語言 | PHP 7.x / 8.x |
| 架構模式 | MVC |
| 資料庫 | MySQL |
| 前端 | HTML5、CSS3、JavaScript |
| 部署環境 | LAMP（Linux + Apache + MySQL + PHP） |
| 版本控制 | Git / GitHub |

## ✨ 主要功能

### 會員系統
- 使用者註冊與登入
- 會員帳戶管理
- 訂餐歷史紀錄
- 個人偏好設定

### 菜單管理
- 菜單分類展示
- 商品詳細說明
- 圖片上傳與展示
- 價格與庫存管理

### 購物流程
- 購物車操作（新增、移除、修改數量）
- 結帳頁面
- 配送地址管理
- 訂單確認

### 付款系統
- 線上支付流程
- 付款狀態追蹤
- 訂單號生成與管理

### 後台管理（可選）
- 菜單上架/下架
- 訂單管理與狀態更新
- 銷售統計

## 🚀 使用方式

### 環境需求
- PHP 7.4+
- MySQL 5.7+
- Apache（支援 mod_rewrite）
- Linux 伺服器（推薦 Ubuntu 20.04）

### 安裝步驟
```bash
# 複製專案
git clone <repository-url>
cd ordering-system

# 建立資料庫
mysql -u root -p < database/schema.sql

# 設定連線資訊
cp config/database.example.php config/database.php
# 編輯 database.php，填入 MySQL 登入資訊

# 設定檔案權限
chmod -R 755 public/
chmod -R 775 public/uploads/

# 啟動 Apache（如未啟動）
sudo systemctl start apache2

# 存取應用
# 開啟瀏覽器進入 http://your-domain/ordering-system
```
